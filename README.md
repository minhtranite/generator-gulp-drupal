# Generator Gulp Drupal
> Drupal theme with Gulp, Babel, UglifyJS, PostCSS , Imagemin, BrowserSync.

## Installation

```bash
npm install -g yo
npm install -g generator-gulp-drupal
```

Then generate your new project:

```bash
cd path/to/your/theme
yo gulp-drupal
```

## Usage

### Dev

```bash
npm run start
```

**Product mode**

```bash
npm run start-prod
```

### Build

```bash
npm run build
```
