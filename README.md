# Generator Gulp Drupal
> Drupal theme with Gulp, Babel, PostCSS, UglifyJS, Imagemin, Browser sync.

## Installation

```bash
npm install -g yo
npm install -g generator-gulp-drupal
```

Then generate your new project:

```bash
cd path/to/your/theme
yo gulp-drupal
```
